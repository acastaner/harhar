Harhar

Humble Avalanche Replayer of HTTP Archives

Download Source Documentation View on GitHub

This tool allows you to generate a HTTP Archive (.har) file using your browser, and replay the content at high volumes using Spirent Avalanche. The program has been tested and works on Windows and Linux (through Mono).
